# Setup Instructions

## Quick Start

Simply open `index.html` in any browser - that's it! The website is ready to use.

The PDF (`amsler-grid.pdf`) is already included and will open automatically when you click the "Open Printable PDF" button.

## What Each File Does

- **index.html** - Interactive calibration tool for displaying grid on monitors
- **amsler-grid.pdf** - Pre-generated PDF for printing at exactly 10 cm × 10 cm
- **styles.css** - Styling for the interactive website

## For Users

**Digital Grid (for data collection):**
1. Open `index.html`
2. Calibrate your monitor using a ruler or credit card
3. Press `F` for fullscreen
4. Use as reference target for camera

**Printed Grid:**
1. Click "Print Grid" button on website
2. Opens the PDF automatically
3. Print at "Actual Size" or "100% scale"
4. Verify size using calibration tools on page 2 of PDF

## Distribution

When distributing this tool, include these files:
- `index.html`
- `styles.css`
- `amsler-grid.pdf`

Users only need to open `index.html` in any browser.

The pre-generated PDF ensures everyone gets the exact same printout (10 cm × 10 cm) regardless of browser or printer differences.
